
# Knowledge graph embedding (KGE) Models 

**Knowledge graph embedding (KGE)** is a technique for representing entities and relations in KGs in low-dimensional semantic spaces. The objective is to preserve the properties of KGs while enabling efficient computation and inference. These embeddings are widely adopted in tasks such as link prediction, knowledge reasoning, and KG completion. 

## 🧾 Data Description

| Model                    | Description                                                                                                 |
|--------------------------|-----------------------------------------------------------------------------------------------------------  |
| TransE                   | Translation-based embedding model that represents relations as vector translations.                         |
| ConvE                    | Convolution-based embedding model that captures nonlinear interactions between entities and relations.      |
| ComplEx                  | Complex-valued embedding model that effectively models asymmetric and symmetric relations.                  |
